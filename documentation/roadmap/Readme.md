# Roadmap

Here on the roadmap you can find the [done](./done), [planned](./planned) or [stale](./hold) features.

For other types of documentation please check the [documentation](../) folder.

## Format

The roadmap format should be something like

```markdown

filename: <Feature>.md

## <Feature>

### Description

### Status

One of this:

- Finished date
- Planned date
- Hold reason

```
